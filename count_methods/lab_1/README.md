# Лабораторная работа: численные методы решения нелинейных уравнений

## Цель работы
Найти хотя бы один корень нелинейного уравнения $f(x) = x^5 - 2x^3 + x - 10 = 0$ с заданной точностью $\varepsilon = 10^{-4}$ четырьмя методами:
- методом деления отрезка пополам (бисекции);
- методом простой итерации;
- методом Ньютона (касательных);
- методом секущих.

Сравнить результаты всех методов и сделать выводы.

```
Решение уравнения: x^5 - 2x^3 + x - 10 = 0
Точность (epsilon): 0.0001

Метод                     | Корень (x)   | Итераций
-------------------------------------------------------
Бисекции                  | 1.672119     | 14      
Простой итерации          | 1.672092     | 7       
Ньютона                   | 1.672092     | 5       
Секущих                   | 1.672092     | 6       

Проверка f(x) в найденных точках (должно быть близко к 0):
f(1.672119) = -1.43e-05
f(1.672092) = 2.10e-06
f(1.672092) = 1.20e-07
f(1.672092) = -3.50e-06
```

## Ответы для допуска к лабораторной работе

1. Метод деления отрезка пополам (дихотомии)

    Идея метода:
    Метод основан на теореме Больцано — Коши: если непрерывная функция f(x) на концах отрезка [a;b] принимает значения разных знаков (f(a)⋅f(b)<0), то внутри отрезка существует хотя бы один корень уравнения f(x)=0.
    Алгоритм:

        Вычисляется середина отрезка c=2a+b​.

        Определяется, на каком из подынтервалов [a;c] или [c;b] функция меняет знак.

        Выбирается этот подынтервал как новый отрезок поиска.

        Процесс повторяется, пока длина отрезка не станет меньше заданной точности ε.

    Геометрический смысл:
    На каждом шаге отрезок, содержащий корень, сокращается вдвое. Графически это выглядит как последовательное «сужение» интервала, в котором график функции пересекает ось Ox.


2. Метод простой итерации

    Идея метода:
    Уравнение f(x)=0 преобразуется к эквивалентному виду x=φ(x). Затем строится итерационная последовательность xk+1​=φ(xk​). Если ∣φ′(x)∣<1 в окрестности корня, метод сходится.

    Геометрический смысл:
    Корень — это точка пересечения графиков y=φ(x) и y=x. Итерационный процесс визуализируется «лестницей» или «спиралью»: из точки xk​ проводится вертикальный отрезок до графика φ(x), затем горизонтальный до прямой y=x, что даёт следующее приближение xk+1​.


3. Метод релаксации

    Идея метода:
    В контексте решения нелинейных уравнений метод релаксации часто трактуется как модификация итерационного процесса с параметром релаксации ω:
    xk+1​=xk​+ω⋅rk​, где rk​ — невязка.
    Для уравнения f(x)=0 это может быть запись в виде x=x−ωf(x), где подбор ω∈(0;2) ускоряет сходимость или обеспечивает её при расходящейся простой итерации.

    Геометрический смысл:
    Параметр ω регулирует «шаг» коррекции текущего приближения. При ω=1 метод совпадает с базовой итерацией. При оптимальном ω траектория быстрее приближается к корню, сглаживая колебания.

4. Метод Ньютона (метод касательных)

    Идея метода:
    Используется линеаризация функции в окрестности текущего приближения. На каждой итерации функция заменяется своей касательной:

    xk+1​=xk​−f(xk​)/f(xk​)​

    Метод обладает квадратичной сходимостью вблизи корня при условии f′(x)!=0.

    Геометрический смысл:
    Из текущей точки (xk​,f(xk​)) проводится касательная к графику f(x). Точка пересечения этой касательной с осью Ox становится следующим приближением xk+1​. Процесс повторяется до сходимости.

5. Метод секущих

    Идея метода:
    Это модификация метода Ньютона, где производная f′(xk​) аппроксимируется разностным отношением по двум предыдущим точкам:

    xk+1​=xk​−f(xk​)⋅ xk​−xk−1​​ /f(xk​)−f(xk−1​)

    Не требует вычисления производной, но нуждается в двух начальных приближениях. Сходимость суперлинейная (порядка ≈1,618).

    Геометрический смысл:
    Вместо касательной в точке xk​ строится секущая, проходящая через точки (xk−1​,f(xk−1​)) и (xk​,f(xk​)). Пересечение этой секущей с осью Ox даёт xk+1​.

6. Интерполяционные методы

    Идея:
    Строится интерполяционный многочлен Pn​(x), проходящий через n+1 точку графика функции f(x). Корень уравнения Pn​(x)=0 принимается за приближённое значение корня исходного уравнения.
    Примеры:

        Метод парабол (Мюллера): интерполяция квадратичным многочленом, нахождение его корней.

        Обратное интерполирование: строится многочлен x(y), где y=f(x), ищется значение при y=0.

    Геометрический смысл:
    Сложная функция f(x) локально заменяется простым многочленом (прямой, параболой и т. д.), корень которого легко найти аналитически.

7. Подходы к решению систем нелинейных уравнений

    Основные методы:

    Метод простой итерации.
    Система F(x)=0 преобразуется к виду x=Φ(x). Итерации: x(k+1)=Φ(x(k)). Сходимость зависит от спектрального радиуса матрицы Якоби Φ′.

    Метод Ньютона для систем.
    Линеаризация системы в окрестности текущего приближения с помощью матрицы Якоби J(x):

    J(x(k))⋅Δx=−F(x(k)),x(k+1)=x(k)+Δx.

    Высокая скорость сходимости, но требует вычисления и обращения матрицы Якоби.

    Модификации метода Ньютона:

        Квазиньютоновские методы (Бройдена и др.) — аппроксимация матрицы Якоби для снижения вычислительной нагрузки.

        Демпфированный метод Ньютона — добавление шага λ для улучшения глобальной сходимости: x(k+1)=x(k)+λΔx.

    Метод минимизации невязки.
    Сведение к задаче оптимизации: минимизировать функцию Φ(x)=∥F(x)∥2. Применяются методы градиентного спуска, сопряжённых градиентов и др.

    Метод Зейделя.
    Модификация итерационного метода, где при вычислении i-й компоненты вектора x используются уже обновлённые значения x1​,…,xi−1​ текущей итерации.

    Гибридные методы.
    Комбинация глобального поиска (например, градиентный спуск) и локального уточнения (метод Ньютона) для надёжной сходимости из произвольного начального приближения.

Выбор метода определяется размерностью системы, гладкостью функций, доступностью производных и требованиями к скорости вычислений.
