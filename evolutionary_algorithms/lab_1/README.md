# Лабораторная работа №1  
## Поиск экстремума функции одной переменной с помощью генетических алгоритмов  
(бинарное представление особей)

---

## Цель работы

Целью данной лабораторной работы является знакомство с основными понятиями теории генетических алгоритмов, а также изучение и практическая реализация классического простого генетического алгоритма с использованием бинарного представления решений. В ходе работы реализуются основные генетические операторы — кроссинговер и мутация — и анализируется их влияние на процесс поиска экстремума функции.

---

## Постановка задачи

Согласно варианту задана функция одной переменной:

f(t) = t^2 - 6t + 5

Необходимо найти экстремум данной функции на отрезке:

t in [0, 8]

Поиск экстремума требуется выполнить с использованием генетического алгоритма с бинарным представлением особей. В процессе работы необходимо визуализировать динамику поиска решения и изменение значений фитнесс-функции.

Для программной реализации используется библиотека **DEAP**, предназначенная для построения и исследования эволюционных алгоритмов.

---

## Теоретические сведения

Генетический алгоритм — это метод оптимизации, основанный на принципах естественного отбора и генетики. Основными элементами генетического алгоритма являются:

- популяция особей (кандидатных решений);
- фитнесс-функция, определяющая качество решения;
- операторы отбора, кроссинговера и мутации;
- итерационный процесс формирования новых поколений.

В данной работе каждая особь представляется в виде бинарной строки фиксированной длины, которая кодирует вещественное значение переменной `t`.

---

## Бинарное кодирование решения

Переменная `t` кодируется бинарной строкой длины `N` бит. Каждая бинарная строка интерпретируется как целое число, которое затем линейно отображается на заданный отрезок `[t_0, t_1]` по формуле:

t = t0 + (D / (2^N - 1)) * (t1 - t0)

где `D` — десятичное значение бинарной строки.

---

## Фитнесс-функция

В качестве фитнесс-функции используется значение исходной функции:

F = f(t)

Так как в данной работе осуществляется поиск **минимума**, фитнесс-функция минимизируется.

---

## Описание алгоритма

Алгоритм работы программы включает следующие этапы:

1. Формирование начальной популяции случайных бинарных особей.
2. Декодирование особей в вещественные значения переменной `t`.
3. Вычисление фитнесс-функции для каждой особи.
4. Отбор особей методом турнирной селекции.
5. Применение оператора одноточечного кроссинговера.
6. Применение оператора мутации (инверсия битов).
7. Формирование нового поколения.
8. Повторение алгоритма до достижения заданного числа поколений.
9. Сбор статистики и визуализация результатов.

---

## Параметры генетического алгоритма

Входными параметрами программы являются:

- размер популяции;
- длина бинарной строки (число бит);
- вероятность кроссинговера;
- вероятность мутации;
- максимальное число поколений.

Значения параметров могут быть изменены пользователем для исследования влияния на сходимость алгоритма.

---

## Результаты работы

В ходе выполнения программы строятся следующие графики:

- график исследуемой функции `f(t)` на заданном интервале;
- динамика перемещения лучших особей по графику функции;
- график изменения лучшего значения фитнесс-функции по поколениям;
- график изменения среднего значения фитнесс-функции популяции.

В результате работы алгоритма было найдено приближённое значение точки минимума функции, близкое к аналитическому решению:
t ≈ 3, f(t) ≈ -4
Небольшая погрешность объясняется стохастическим характером генетического алгоритма.
