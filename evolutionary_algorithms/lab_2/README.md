# Лабораторная работа №2  
## Поиск экстремума функции двух переменных с помощью генетических алгоритмов  
(вещественное представление особей)

---

## Цель работы

Цель лабораторной работы — модификация классического простого генетического алгоритма для задачи многомерного поиска экстремума функции двух переменных с использованием **вещественного представления решений**. В работе изучаются основные операторы вещественной рекомбинации (кроссинговер, мутация) и методы селекции, а также визуализируется процесс поиска экстремума на плоскости X1×X2.

---

## Постановка задачи

Дана функция двух переменных:

f(x1, x2)


> Необходимо найти её экстремум на области:

x1 ∈ [a1, b1], x2 ∈ [a2, b2]


с помощью генетического алгоритма с вещественным представлением особей. В процессе работы требуется визуализировать:

- линии уровня функции на плоскости X1×X2;
- движение особей по плоскости относительно линий уровня функции;
- графики лучшего и среднего значения фитнесс-функции популяции;
- координаты точки с наилучшей фитнесс-функцией;
- значение фитнесс-функции для этой точки.

---

## Теоретические сведения

Генетический алгоритм (ГА) — это метод оптимизации, основанный на принципах естественного отбора и генетики. Основные элементы:

- **Популяция особей** — каждый кандидат решения представлен в виде списка вещественных чисел `[x1, x2]`.
- **Фитнесс-функция** — значение функции `f(x1, x2)`; в данной работе производится **минимизация**.
- **Операторы**:
  - **Селекция** — турнирная или ранжированная;
  - **Кроссинговер** — однородный или многоточечный;
  - **Мутация** — добавление случайного шума (гауссовская мутация).
- **Итерационный процесс** — повторение генераций до достижения максимального числа поколений.

---

## Входные параметры программы

- размер популяции;
- вероятность кроссинговера;
- вероятность мутации;
- максимальное число поколений.

---

## Реализация в Python с DEAP

Пример функции и границ поиска:

```python
def f(x1, x2):
    return x1**2 + x2**2 - 4*x1 - 6*x2 + 13  # минимум в (2,3)

X1 ∈ [-5, 5]
X2 ∈ [-5, 5]
```

## Визуализация результатов

> Линии уровня функции показывают форму поверхности.

> Красные точки и линия — движение лучших особей по плоскости.

> Графики динамики фитнесс-функции отображают:

> лучшее значение в поколении,

> среднее значение популяции.

> Координаты лучшей особи на каждом поколении и её значение фитнесс-функции выводятся в консоль.


## Выводы

> Генетический алгоритм с вещественным представлением особей эффективно решает задачи многомерной оптимизации.

> Однородный кроссинговер и гауссовская мутация обеспечивают стабильное продвижение к экстремуму.

> Турнирная селекция поддерживает разнообразие популяции.

> Визуализация линий уровня и движения особей позволяет наглядно наблюдать процесс поиска экстремума.
